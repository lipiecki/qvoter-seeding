# Influence Maximization in Competitive Complex Contagions
This repository contains codes and network data for simulations designed to evaluate the efficiency of selected network seeding stratagies in maximizing influence within the q-voter model.

Dataset sources:
- [SNAP Facebook network](https://snap.stanford.edu/data/ego-Facebook.html)
- [Facebook page network of politicians](https://networkrepository.com/fb-pages-politician.php)
